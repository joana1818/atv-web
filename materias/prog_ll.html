Programação II em linguagem C

  • Introdução à Linguagem C:

    • C é uma linguagem de programação de nível intermediário criada por Dennis Ritchie em 1972 na Bell Labs.
    • É amplamente usada para sistemas operacionais, drivers de dispositivos, compiladores e outras aplicações de baixo nível.

  • Compilação e Execução:

    • Um programa em C é escrito em um editor de texto e salvo com a extensão ".c" (por exemplo, "meu_programa.c").
    • Para compilar, use um compilador como GCC (GNU Compiler Collection) digitando gcc meu_programa.c -o meu_programa.
    • Isso produz um arquivo executável chamado "meu_programa" (ou o nome especificado com a opção -o).
    • Para executar, basta digitar ./meu_programa no terminal.

  • Variáveis e Tipos de Dados

    • As variáveis em C devem ser declaradas com um tipo antes de serem usadas.
    • Exemplos de tipos de dados incluem int, float, char, etc.
